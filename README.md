# Photo-Download

#版本：V0.0.1

由於喜歡動漫並且也喜歡相關插圖,某一天在網上無聊逛逛看見了**pixiv**,並且發現喜歡的插圖於是就收藏了一番,礙於久了手會有點殘所以就想懶一點,為什麼我不讓他自己來呢？


## 目前功能

* 輸入照片ＩＤ

* 儲存此頁面插圖

* 儲存此頁面其他六張插圖

* 保存在images


## 使用方法
#####1.下載檔案：
直接下載或是複製網址使用git clone 
#####2.安裝必要插件：
使用終端機,在檔案目錄下輸入 npm install
#####3.使用：
1.編輯app.js,輸入單一照片ＩＤ

2.終端機輸入 npm start 就會一直開著

3.只要ＩＤ變更程式會重新執行

## 目前狀況
並不是每一張圖片都能成功,失敗率高

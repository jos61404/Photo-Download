# Photo-Download

##版本：v0.1.1

由於喜歡動漫並且也喜歡相關插圖,某一天在網上無聊逛逛看見了**pixiv**,並且發現喜歡的插圖於是就收藏了一番,礙於久了手會有點殘所以就想懶一點,為什麼我不讓他自己來呢？

### 執行環境 :

* Node.js v7.1.0 （官方網站：<https://nodejs.org/en/>）

### 目前功能 :

* 輸入插畫師ID (不知道請看網址)

* 保存全部作品

* 預設保存在images(可定義)

### 使用方法 :
#####1.檔案下載：

    直接下載或是複製網址使用git clone

#####2.必要程式安裝：

    1.下載Node.js安裝程式，並且安裝

    2.使用終端機,在檔案目錄下輸入 npm install （自動安裝套件）


#####3.使用說明：

    1.編輯config.json

    2.輸入帳號與密碼

    3.終端機輸入 node app.js
